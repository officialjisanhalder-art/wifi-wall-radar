## 🧠 The Science Behind It

How can WiFi see through walls? 

Traditional WiFi devices communicate by sending electromagnetic waves. When these waves encounter an object (like a wall or a human body), they undergo **scattering, reflection, and attenuation**.

- **Channel State Information (CSI)**: While RSSI (Received Signal Strength Indicator) only measures the total signal strength, CSI measures the amplitude and phase shifts of *individual subcarrier frequencies* in OFDM transmissions.
- **Multipath Signature**: Human bodies consist of ~70% water, which highly absorbs/reflects 2.4GHz & 5GHz signals. When a person moves, they create unique dynamic phase shifts in the subcarrier waves.
- **Doppler Shift**: By analyzing these phase changes over time, algorithms can calculate the target's speed, breathing rate, and even 2D coordinate locations behind obstacles.

## 🚀 Key Features

- **Spatial Sweep Canvas** — Sweeping radar visualization calculating human coordinates behind drywalls, bricks and concrete.
- **Multipath Vector Tracking** — Dotted vector projections mapping real-time signal reflections from the transmitter to the target.
- **Oscilloscope CSI Wave Plotter** — Continuous subcarrier sine wave generator mapping signal amplitude distortions.
- **Parametric Controls** — Adjust carrier frequencies (2.4GHz, 5.0GHz, 60GHz), change subcarrier resolution and swap MIMO configurations.
- **Terminal Diagnostics logs** — Simulates raw frames feedback showing Doppler shifts, SNR readings, and calculations.
